# Progress Log - 2026-01-19

## Codebase Patterns

### Laravel Konventionen
- Laravel 12 mit Inertia.js + React 19
- Controller-Methoden returnen `Inertia::render()` oder `RedirectResponse`
- Form Requests für Validation (in app/Http/Requests/)
- Policies für Authorization (auto-discovery in Laravel 12)
- Pest v4 für Tests

### Frontend Konventionen
- TypeScript strict mode
- Wayfinder für typisierte Routes (`php artisan wayfinder:generate`)
- Tailwind CSS v4 (CSS-first config mit @theme)
- shadcn/ui Components

### Deck Feature Struktur
- Models: `Deck`, `DeckZone`, `DeckCard` in app/Models/
- Relations: Deck -> GameFormat -> Game
- DeckCard -> UnifiedPrinting -> UnifiedCard
- Services: DeckValidationService, DeckbuilderService

### Wichtige Commands
- `php artisan test --filter=Deck` - Deck-Tests
- `npm run build` - Frontend Build
- `vendor/bin/pint --dirty` - Code Style
- `php artisan wayfinder:generate` - TypeScript Routes
